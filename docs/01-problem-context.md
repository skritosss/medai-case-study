# Problem Context

Clinicians often need to make decisions under incomplete information, time pressure, and strict safety expectations. AI-assisted products in this domain must be designed as decision-support tools, not as black-box replacements for professional judgment.

The product context behind MedAI involved a specialized pediatric workflow where structured guidance, clear inputs, and traceable outputs were important.

## Product Questions

- What information does a clinician need before making a decision?
- Which data fields are mandatory?
- Which edge cases should stop the workflow and request expert review?
- How can the interface make uncertainty and escalation paths visible?
