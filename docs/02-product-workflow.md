# Product Workflow

```text
case context
  -> structured data entry
  -> validation
  -> decision-support workflow
  -> explanation / review step
  -> clinician review
  -> documented outcome
```

## Workflow Principles

- The user must understand why an output appears.
- Missing or ambiguous data should be treated as a workflow state, not as a hidden error.
- The product should support professional review rather than remove it.
